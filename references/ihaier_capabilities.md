# ihaier：能力与场景

- **能力**：ihaier 内可执行的操作或入口（读消息列表、消息详情、打开某功能模块等）。
- **场景**：由若干能力组合而成的用户需求（如「谁找我」= 读消息列表 + 可选读消息详情）。

---

## 一、能力（ihaier 内可执行的操作）

以下为 ihaier 左侧导航及主界面可触达的**能力**，实现方式均为：激活 ihaier 窗口 → 截图 → vision 定位入口/区域 → click（或按键），必要时再截图+vision 读内容。

| 能力 | 说明 | 入口/方式 |
|------|------|------------|
| **读取消息列表** | 看到中间栏会话列表（头像、会话名、摘要、时间、未读红数字） | 左侧点「消息」进入消息视图；当前若已在消息页则直接截图即可。 |
| **消息详情** | 看到某会话右侧完整内容（谁发的、正文、附件、输入框） | 在消息列表中 click 某一条会话，右侧即显示详情；再截图+vision 可提取「谁、什么事」。 |
| **视频会议** | 进入视频会议模块 | 左侧导航点「视频会议」。 |
| **云文档** | 进入云文档模块 | 左侧导航点「云文档」。 |
| **会议室预约** | 进入会议室预订 | 左侧导航点「会议室预订」。 |
| **工作台** | 进入工作台 | 左侧导航点「工作台」。 |
| **审批** | 进入审批模块 | 左侧导航点「审批」。 |
| **日历** | 进入日历 | 左侧导航点「日历」。 |
| **任务** | 进入任务 | 左侧导航点「任务」。 |
| **邮箱** | 进入邮箱 | 左侧导航点「邮箱」。 |
| **机器人 / 收藏 / II服务台 / 更多** | 对应功能入口 | 左侧导航点击对应项。 |
| **搜索** | 调出搜索（如搜联系人、应用） | 顶部「Q 职业 (Ctrl+K)」或快捷键 Ctrl+K。 |
| **发消息（输入并发送）** | 在某个会话里输入文字并发送 | 进入该会话后，在右侧底部**点击输入框** → **Ctrl+V 粘贴**（执行前设置剪贴板）→ **Enter 发送**或点击**右下角发送按钮**。见 plan `example_ihaier_send_message.json`。 |

**通用前置**：聚焦 ihaier 窗口（`window_tool activate "办公平台"`，任务管理器中主窗口名为「办公平台」；或 `activate_process iHaier2.0`），再截图+vision 识别上述入口位置并 click。

## 二、场景（组合能力形成的用户需求）

场景是**对能力的组合使用**，不是能力本身。

| 场景 | 用到的能力 | 说明 |
|------|------------|------|
| **获取未读消息 / 谁找我 / 是否有人找我** | 读取消息列表（+ 可选：消息详情） | 看左侧消息红数字、中间列表红数字与加粗；若要「什么事」再点进会话用「消息详情」。 |
| **获取「我的」最新消息** | 读取消息列表 + 消息详情 | 中间列表最上方置顶，第一条多为「我的」；点进后右侧对话框最下面一条为最新消息。见 plan `example_ihaier_my_latest_message.json`。 |
| **发消息给某人** | 搜索或联系人 + 进入会话 + 发消息（输入并发送） | 找联系人/会话 → 输入内容 → 发送。 |
| **查看审批** | 审批 | 进入审批模块后截图+vision 看列表或详情。 |
| **预约会议室** | 会议室预约 | 进入会议室预约后按界面操作。 |

规划或 run_plan 时：先选**能力**（要读消息列表还是进云文档），再组合成**场景**（例如先「读取消息列表」再 vision 判未读，即「谁找我」场景）。

## 三、计划与引用

- 获取未读消息、列出未读会话：`plans/example_ihaier_check_messages.json`、`plans/example_ihaier_who_contacted_me.json`（读取消息列表 + vision）。
- 获取「我的」最新消息：`plans/example_ihaier_my_latest_message.json`（激活后先**点击中间列表最上方圆形头像（我的）**，再截图读最新一条；勿点其下长方形会话条目。若提示词中明确当前用户姓名如周小帅，vision 可据此确认「我的」即该用户会话）。
- 发消息：`plans/example_ihaier_send_message.json`（激活 ihaier → 点击输入框 → 粘贴剪贴板 → Enter 发送；执行前需设置剪贴板为要发送的内容）。
- 其他能力（视频会议、云文档、会议室预约等）：暂无独立 plan，统一走「激活 ihaier → 截图 → vision 定位左侧入口 → click」，后续可按需加命名 plan。

私域与布局细节见 [private_domains.md](private_domains.md)「办公平台 · ihaier」。
