# UI 与智商阶段

## 科幻主体

- **风格**：类似贾维斯/星期五的科幻主体，持续自我完善。
- **载体**：可选 Web 本地页或桌面内嵌；前端骨架放在 `assets/`，随进化迭代。

## 智商阶段

用「阶段」对应不同难度，便于不同用户（如不同年龄/经验）使用电脑完成不同难度的事：

| 阶段   | 含义（示例）           |
|--------|------------------------|
| 初级   | 简单点击、打开关闭应用 |
| 中级   | 多步流程、表单填写     |
| 高级   | 自动化脚本、条件分支   |

阶段可与「当前使命」「任务追踪」联动展示，让用户与智能体都清楚当前能力层级。

## 与状态的关系

- UI 可读取 `state/current_mission.json` 展示「当前在干什么」。
- 日志可被 UI 展示为「行为历史」，便于溯源。
