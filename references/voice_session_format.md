# 语音会话文件格式

语音对话与 CC 时，会话记录写入 `runtime/voice_sessions/voice_YYYYMMDD_HHMMSS.md`。

## 格式约定

| 标记 | 含义 |
|------|------|
| `## 你` | 用户输入（ASR 转文字） |
| `## CC` | CC 的回复区块 |
| `--日志` | CC 执行日志行（CC 写入时每行前加此前缀） |
| `--已完成` | CC 处理完成标记（CC 在最终回答后加此行） |

## CC 写入要求

CC 收到提示词后需：
1. 实时将执行日志写入该文件（每行前加 `--日志`）
2. 最终结果写在 `## CC` 区块内
3. 完成后加一行 `--已完成`

## 示例

```markdown
# 语音会话 2025-03-04 12:34:56

## 你
你好呀

## CC
--日志
正在读取 references...
--日志
解析完成
你好！有什么可以帮你的？
--已完成
```
